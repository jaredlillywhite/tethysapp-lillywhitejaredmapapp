Welcome to Jared's Map App!

By Jared Lillywhite, CEEN 514 2/24/2018

Assignment Requirements:

Create 4 links on the sidebar:

  • Home
  • Map View
  • Data Services
  • About

Each of these links should open a new page in the main view.

  • Home should open a "home page" with the name of the class, name of your app, date, assignment number and main assignment requirements listed.
  • Map View should include an ArcMap Javascript API map with two data sets from your ArcMap Geoserver2 server, toggle buttons to turn off and on a raster layer and vector layer, and popup bubbles with information about the features in your map
  • Data Services will be a page that talks about the data services you have used in your map, how you created the data, where you are hosting the data, what data is included, and any additional data about your data. Also include a link to the geoserver2 data view page for your data.
  • About - the about page will include a picture of you, a link to send you an email, and a little bit about who you are.
